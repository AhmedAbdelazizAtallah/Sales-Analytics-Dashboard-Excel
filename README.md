# 📊 Sales Analytics Dashboard (Microsoft Excel)

An interactive Sales Analytics Dashboard developed in Microsoft Excel to analyze sales performance, profitability, customer behavior, shipping operations, and regional trends.

---

## 📌 Project Overview

This project demonstrates how Excel can be used as a Business Intelligence tool to transform raw sales data into an interactive dashboard for decision-making.

The dashboard allows users to explore sales performance across products, customers, regions, and shipping methods using dynamic filters and visualizations.

---

# 🎯 Objectives

- Analyze overall sales performance
- Evaluate profitability
- Identify top-selling products
- Compare regional performance
- Analyze customer segments
- Monitor shipping performance
- Track sales trends over time
